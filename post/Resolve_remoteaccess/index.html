解决Exsi创建虚拟机无法远程问题

2021-05-06 Views 系统运维

在Exsi中创建虚拟机做测试，Esxi版本是6.5，虚拟机系统是Win Server 2012，已手动分配ip，和使用物理机是同网段地址。问题：无法远程访问

问题分析：

在为其安装好系统之后，无法使用物理机进行远程访问，大多数原因是网络不通、防火墙拒绝访问、远程设置未开启等，在排查之后，原因是远程设置未开启导致的无法远程访问。

问题解决：

打开设置-系统和安全-远程设置，将“远程桌面”选项中，设置为“允许远程连接到此计算机”，并取消勾选“仅循序运行使用网络级别身份验证的远程桌面计算机连接”，应用保存，测试访问，OK！解决！
